# 🌸 Auroraa ♡ 
> **The ultimate mood-sync player.**

Auroraa is a web app that listens to your feelings and picks the perfect vibe from my personal Spotify collections. Just tell Auroraa how you're feeling, log in, and let the music take over.

---

## 🚀 How to Run
To start vibing, just follow these simple steps:

1. Clone the project
2. Install the essentials "pip install flask spotipy python-dotenv " 
3. Create a .env file in the folder and add your Spotify Client ID and Secret (from the Spotify Developer Dashboard).
4. Start the app. "python app.py"

## 🎧 How to Listen
1. Open Spotify: Make sure Spotify is open on your phone or computer.
2. Log In: Click the login button in Auroraa to connect your account.
3. Set the Mood: Type your mood (happy, sad, love, dance, or fun).
3. Enjoy: Auroraa will automatically start playing my curated playlists directly on your Spotify!
